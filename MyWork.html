Dice Game

This website is hosted on cloudflare. HtML,CSS and JavaScript has been used to build this.

Technologies Used

  • HTML
  • Css
  • JavaScript

Motivation

I tend to take a decision from rolling a dice which happen to inspire me to make this. Big step for me that time as i had just started learning JavaScript. Was very proud of myself after completing it.

Project difficulties

Since i was new to JavaScript, i had to learn the concept of DOM in website and had to go through many tutorials. The main issue was the condition that would decide whether player 1 won or was it player 2.

My Solution

To solve this struggle, i decided to kick it old school with some good of object oriented design and analysis. Following a planning structure, i was able to improve my work efficiency and reach my set sprints.

Notable Features

  • Designed completely through CSS without any external libraries.
  • Used Vanilla JavaScript

Motivation

Odin Project inspired me to learn about Flexbox. Even though it is a new way to layout, it has been widely used by many websites and is extremely convenient. This is one of the projects i made from the website Odin Project.

Project difficulties

I thought i had the complete grasp of margin, padding, etc,. However, flexbox proved me wrong. It really challenges you and pushes the boundaries which really assists you to understand better, how everything is laid out. The most confusing part is making several div containers so as to group all the elements in one box to align with the other divs.

My Solution

I first wrote all the html and decided which elements need to be in one container which made it alot easier before taking action. In addition to that, i added the all the elements on the left in one conatiner and vice versa. After i just had to the same thing for the other elements which helped me to keep this code as simple and neat as possible.

Notable Features

  • Flexbox has been used.
  • CSS for the Design
  • Made the image responsive.
